# AnyDesk安装研判

## 目标
判断AnyDesk远程访问软件安装告警是真实攻击行为，还是正常业务使用。

## 范围
- AnyDesk安装
- AnyDesk静默安装
- 远程访问软件部署
- 可疑RMM工具安装
- 持久化机制

## 输入
| 参数 | 说明 |
|------|------|
| `${alert:name}` | 告警名称 |
| `${alert:severity}` | 告警等级 |
| `${alert:hostName}` | 主机名 |
| `${alert:processPath}` | 安装进程路径 |
| `${alert:commandLine}` | 命令行参数 |
| `${alert:filePath}` | 安装文件路径 |
| `${alert:parentProcess}` | 父进程信息 |
| `${alert:user}` | 执行用户 |

## 输出
| 参数 | 说明 |
|------|------|
| `${alert:triageResult}` | 研判结果 |
| `${alert:triageCoT}` | 研判思维链 |

## 工具
- **EDR工具**: `${edr-mcp}`
  - `${edr-getProcessTree}` - 获取进程树
  - `${edr-getFileHash}` - 获取文件哈希
  - `${edr-getRegistryValue}` - 获取注册表值
- **SOC工具**: `${soc-mcp}`
  - `${soc-getAlert}` - 获取告警信息
  - `${soc-getAsset}` - 获取资产信息

## 流程

### 正报判断

#### 1. 安装方式分析
- **1-1.** 分析命令行参数
  - 从 `${alert:commandLine}` 识别静默安装参数
  - 检查是否包含 `--silent`、`--install`、`/S` 等静默参数
  - 识别是否配置自动启动：`--start-with-win`
  - 检查是否预设密码：`--set-password`

- **1-2.** 分析父进程
  - 从 `${alert:parentProcess}` 识别调用来源
  - 检查是否为漏洞利用后的异常进程（如Java、Web服务）
  - 识别是否为注入进程创建
  - 检查是否为Metasploit/Cobalt Strike进程

- **1-3.** 分析安装路径
  - 从 `${alert:filePath}` 判断安装位置
  - 检查是否在临时目录执行安装程序
  - 识别是否为可疑的安装源路径

#### 2. 配置特征分析
- **2-1.** 分析注册表配置
  - 通过 `${edr-getRegistryValue}` 获取AnyDesk配置
  - 检查是否配置了自动启动
  - 识别是否设置了访问密码
  - 检查是否禁用了用户交互

- **2-2.** 分析服务配置
  - 检查是否创建了AnyDesk服务
  - 识别服务启动类型是否为自动
  - 检查服务是否立即启动

#### 3. 关联行为分析
- **3-1.** 分析前置告警
  - 通过 `${soc-getAlert}` 获取该主机近24小时告警
  - 检查是否存在漏洞利用告警
  - 识别是否存在凭证转储告警
  - 分析是否存在其他持久化告警

- **3-2.** 分析网络连接
  - 检查安装后是否立即建立远程连接
  - 识别连接来源IP是否可疑
  - 分析登录IP是否与已知C2重合

- **3-3.** 分析时间线
  - 检查是否在非工作时间安装
  - 分析从漏洞利用到安装的时间间隔

### 误报判断

#### 1. 业务场景分析
- **1-1.** 分析安装来源
  - 检查是否从公司软件仓库安装
  - 识别是否为IT部门推送的安装
  - 判断是否为用户主动安装

- **1-2.** 分析资产属性
  - 通过 `${soc-getAsset}` 获取主机资产信息
  - 判断是否为远程办公用户终端
  - 检查是否为需要远程支持的设备

#### 2. 历史行为分析
- **2-1.** 分析历史安装
  - 通过 `${soc-getAlert}` 获取历史AnyDesk告警
  - 判断该主机是否有AnyDesk使用历史
  - 检查是否存在正常的远程支持工单

- **2-2.** 分析授权状态
  - 检查AnyDesk是否在公司授权软件列表
  - 验证是否在IT服务台记录中

#### 3. 安装特征分析
- **3-1.** 分析安装来源
  - 通过 `${edr-getFileHash}` 获取安装文件哈希
  - 检查是否为官方版本的哈希值
  - 识别是否为修改版或捆绑恶意软件
